La base de datos corresponde al archivo "postgresDB.sql", la cual corresponde a una base de datos desarrollada en postgreSQL. De todas formas, se adjunta dentro de la carpeta "Proyecto/otros/Scripts DB/" los archivos .sql de creación de tablas y de inserción de datos.
Se debe utilizar como DBMS el sistema pgAdmin para su correcta ejecución.

1. npm init -y
2. Instalar paquetes:
	-- npm install express
	-- npm install pg
	-- npm install jsonwebtoken bcrypt express-validator
